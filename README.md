# Oversight — Beta Release Website

Product launch website for [Oversight](https://the-painters-product.vercel.app), an AI chatbot conversation quality assurance tool by The pAInters (CSC491, University of Toronto).

Built with [Docusaurus 3](https://docusaurus.io/).

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deployment

Deployed automatically to Vercel on push to `main`.
